Bienvenido a la API de DecaSport

Esta API proporciona acceso a varios recursos para gestionar usuarios y productos de la tienda online DecaSport.

Endpoints Disponibles:

  • /users - GET: Obtiene todos los usuarios
  • /users - POST: Crea un nuevo usuario
  • /mail - PUT: Actualiza un usuario existente por su dirección de correo electrónico
  • /products - GET: Obtiene todos los productos
